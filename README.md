# react-calcalutor

The Simple Calcalutor using react js 

Download the Files 
To run the project using these command
```
npm start
```
The local server adderss is 
```
 http://localhost:3001
 ```
 To activate the JSON server 
 ```
 npm run server
 ```

 ```
 http://localhost:3000/tasks
 ```
